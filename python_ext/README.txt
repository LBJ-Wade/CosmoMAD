PyCosmoMAD is a python wrapper for CosmoMAD.

Installing:
  Assuming the CosmoMAD C library has been installed and can
  be found in the LDPATH, simply type:
  $ python setup.py install

Usage:
  just import package. All CosmoMAD functions are supported,
  and they can be called by just omitting the csm_ prefix.
  See test/test_csm.py for details.
